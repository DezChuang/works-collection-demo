JS | Color Guessing Game

  • 作品連結
    • https://codepen.io/dezchuang/pen/aJLGME

  • 作品介紹

    • 2016年底初學網頁技術時的練習，透過基本的JS操作DOM來實作「猜RGB實際顏色」的小遊戲。

    遊戲分為EASY/HARD兩種模式，猜錯顏色的話，色塊會消失，猜對的話全部色塊會變為當前RGB顏色，可以按PLAY AGAIN一直玩。


  • 使用技術
    • HTML
    • CSS
    • Javascript
