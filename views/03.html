React | Pokemon Dashboard

  • 作品連結
    • http://dezpokedashboard.herokuapp.com/

  • 作品介紹

    • 在Udemy上學習的第一堂React課程，跟著Tim的教學，練習用Pokemon API與React技術實作一個Pokemon圖鑑。

    其中實作的功能有：

    • 每頁最多顯示筆數
    • 利用React modal呈現每隻Pokemon的詳細資料
    • 利用React chartJS呈現能力曲線
    • 底部分頁功能

    從實作此作品學習到React的模組化想法與ES6方便的寫法，最後為了Demo作品，也使用Webpack將專案deploy到Heroku上，由於是使用Heroku的免費專案，第一次連到網址時會需要等一會。


  • 使用技術
    • React
    • React-Bootstrap
    • react-chartjs
    • Webpack
    • Heroku
    • Pokemon API
